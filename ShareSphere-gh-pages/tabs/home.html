ShareSphere is a modern, easy to use, and feature rich cross-game social network for Roblox.

About ShareSphere

ShareSphere is a cross-game social network for Roblox, but what does that really mean?

In simpler terms, it's twitter, but for roblox. In each game with ShareSphere installed, users are able to open the ShareSphere window and share posts. Users can choose to share posts globally, with all games that have ShareSphere installed, or locally, just to the game they're currently in.

These posts can include pictures, invite links(to server or game), or donation requests. And with new features in the works, users will soon be able to include content such as polls, and catalog items.

Posts on ShareSphere can also be promoted. Users can promote their post, sending it to the top of the global or local feed for a robux charge. This can be beneficial to many groups including content creators and game developers. For example, a content creator could promote a post to get players to join the server they're in, or game developers can promote posts to advertise their games.

Installation

Installing ShareSphere is easy, but there are some important steps and options, so make sure to read the guide thoroughly.

Step 1.

Decide whether you want to use the open source, fully customizable version of ShareSphere†, or the more locked down version of ShareSphere, which enables more features for both you the developer, and for the users.

If you chose to use the first version

You can simply download this Roblox model and set up ShareSphere for yourself.

If you chose to use the second version

Step 2.

Insert this Roblox model into your game. (The model should be inserted into ServerScriptStorage, the loader sorts all components into their places when the server starts.)

Step 3.

Find the "Configuration" object inside of the script called "ShareSphere Loader" and paste your customization code into the stringValue called "customizationCode".

Step 4.

Allow HTTP Requests
Allow Third-Party-Teleports
Allow Third-Party-Sales
Enable Studio Access to API Services(optional, highly recommended)

That's it! ShareSphere is all set up.

Local feed promotion's price is based on several factors such as average players, etc. Post promotion for posts with extra content such as invite links may cost more. Post promotion can cost from 15 to 250.
†Features may be limited or different between the two versions of ShareSphere. ShareSphere Secure gives access to global feed posting, ShareSphere Open only has support for local feed posting.
